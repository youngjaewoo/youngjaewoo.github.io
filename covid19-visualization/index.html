Get COVID19 data

COVID data (cleaned) is available here at Github. The data gets updated by Johns Hopkins University. Import the data.

using CSV, DataFrames, DataFramesMeta, Statistics, FreqTables, StatsBase, Dates, Plots.PlotMeasures
using Plots, StatsPlots; gr(size=(800,300), bottom_margin = 4mm, left_margin = 2mm)
ENV["LINES"] = 20
ENV["COLUMNS"] = 160

datasource = "https://raw.githubusercontent.com/CSSEGISandData/COVID-19/master/csse_covid_19_data/csse_covid_19_time_series" # get raw contents
timedat0 = CSV.File(download("$datasource/time_series_covid19_confirmed_global.csv")) |> DataFrame # COVID19 confirmed numbers
timedat1 = CSV.File(download("$datasource/time_series_covid19_deaths_global.csv")) |> DataFrame ; # COVID19 death numbers

Dictionary mapping country names are used for relabeling column names for the generated results.

dict_country = Dict("Korea, South"=>"Kor", "US"=>"US", "Italy"=>"Ita", "Spain"=>"Spa", "Iran"=>"Ira")

China was excluded for various reasons that makes in incomparable to above countries.

Examine case numbers by country

Extract data for each country using user-defined functions.

function get_data_by_country(timedat, countryname; dict_country=dict_country)
    dformat = DateFormat("m/d/y")
    d = Date.(map(x->string(x, "20"), string.(names(timedat)[5:end])), dformat)

    i = findall(map(x->occursin(countryname, x), timedat[!,2]))[] # 157
    n = convert(Array, timedat[i,5:end])
    timedat_by_country = DataFrame(Date=d, N=n) ;
    rename!(timedat_by_country, :N => Symbol(string("N_", dict_country[countryname])))

    return timedat_by_country
end

# collect data from a list of countries
list_countries = ["Korea, South", "US", "Italy", "Spain", "Iran"]

# Number of confirmed cases
dd0 = get_data_by_country(timedat0, list_countries[1]; dict_country=dict_country)
for cn in list_countries[2:end]
    dd = get_data_by_country(timedat0, cn)
    dd0 = join(dd0, dd, on=:Date)
end

# Number of deaths
dd1 = get_data_by_country(timedat1, list_countries[1]; dict_country=dict_country)
for cn in list_countries[2:end]
    dd = get_data_by_country(timedat1, cn)
    dd1 = join(dd1, dd, on=:Date)
end

Visualize time series.

# visualize the counts per data type
p0 = @df dd0 plot(:Date, [:N_Kor, :N_US, :N_Ita, :N_Spa, :N_Ira], label=["Korea" "US" "Italy" "Spain" "Iran"], ylabel="N.Confirmed")
p1 = @df dd1 plot(:Date, [:N_Kor, :N_US, :N_Ita, :N_Spa, :N_Ira], label=["Korea" "US" "Italy" "Spain" "Iran"], ylabel="N.Deaths")
plot(p0, p1, layout=(1,2), legend=:topleft, xlabel="Date")

Korea is different from other countries for both confirmed and death counts. It's largely attributed to their "Suppresion" method towards COVID19. It's a bit scary that Italy, Spain, and US are following a similar pattern. (Remember I'm only looking at few countries so this is not generalizable..)

# visualize the standardized version for comparisons
p0 = @df dd0 plot(:Date, zscore.([:N_Kor, :N_US, :N_Ita, :N_Spa, :N_Ira]), label=["Korea" "US" "Italy" "Spain" "Iran"], ylabel="Standardized N.Confirmed")
p1 = @df dd1 plot(:Date, zscore.([:N_Kor, :N_US, :N_Ita, :N_Spa, :N_Ira]), label=["Korea" "US" "Italy" "Spain" "Iran"], ylabel="Standardized N.Deaths")
plot(p0, p1, layout=(1,2), legend=:topleft, xlabel="Date")

The counts were z-transformed to be better comparable across countries. Perhaps something better could have been done but this was quick to do... Here's some thoughts.

  • Similar curve flattening observed for Korea as the above.
  • Growth curvature for number of confirmed cases seems to be largest for US > Spain = Italy > Iran
  • Number of deaths increases linearly in Korea. However, this is exponential for other countries. This could be due to 1) time phase difference or 2) method of disease control difference. Can't tell which from quick look.
  • The death growths in Spain and US are steeper at a later date. Kinda scary.. but this may be due to US and Spain are in the early states of COVID19 attack.

Construct cumulative sum from day0 of each country

Cumulative sums are used to display the total sum of data as it grows with time. I thought this was a useful measure to examine the total contribution so far given measure against time. To make it comparable across countries, I designated day 0 following two definitions.

  • day0 = day positive case observed
  • day0 = day after 100 cases

Day0 = Date after 100 cases for each country

Let's look at the cumulative sum using the below thresholds.

  • 100 Confirmed cases
  • 50 Death cases
function get_cumsum_data_by_country(timedat, countryname; dict_country=dict_country, day0after=100)
    dformat = DateFormat("m/d/y")
    d = Date.(map(x->string(x, "20"), string.(names(timedat)[5:end])), dformat)

    i = findall(map(x->occursin(countryname, x), timedat[!,2]))[] # 157
    n = convert(Array, timedat[i,5:end])    

    # defining day0
    i0 = findfirst(n .>= day0after) # day0after = 100 designates day0 as the day first observing 100 cases
    days = collect(1:length(n)) .- i0
    n_cumsum = cumsum(n[i0:end])

    timedat_by_country = DataFrame(Date=d, days=days, N=cumsum(n)) ;
    rename!(timedat_by_country, :N => Symbol(string("N_", dict_country[countryname])))

    return timedat_by_country
end

# collect data from a list of countries
list_countries = ["Korea, South", "US", "Italy", "Spain", "Iran"]

# Number of confirmed cases
nconfirm_thr = 100
dd0c = get_cumsum_data_by_country(timedat0, list_countries[1]; day0after=nconfirm_thr)[:,2:3]
for cn in list_countries[2:end]
    dd = get_cumsum_data_by_country(timedat0, cn; day0after=nconfirm_thr)[:,2:3]
    dd0c = join(dd0c, dd, on=:days, kind=:outer)
end

# Number of deaths
ndeath_thr = 50
dd1c = get_cumsum_data_by_country(timedat1, list_countries[1]; day0after=ndeath_thr)[:,2:3]
for cn in list_countries[2:end]
    dd = get_cumsum_data_by_country(timedat1, cn; day0after=ndeath_thr)[:,2:3]
    dd1c = join(dd1c, dd, on=:days, kind=:outer)
end

Visualize.

p0 = @df @where(dd0c, :days .> -1) plot(:days, [:N_Kor, :N_US, :N_Ita, :N_Spa, :N_Ira], label=["Korea" "US" "Italy" "Spain" "Iran"], ylabel="CumSum N.Confirmed", title="Day0 = After 100 Confirmed Cases")
p1 = @df @where(dd1c, :days .> -1) plot(:days, [:N_Kor, :N_US, :N_Ita, :N_Spa, :N_Ira], label=["Korea" "US" "Italy" "Spain" "Iran"], ylabel="CumSum N.Confirmed", title="Day0 = After 50 Death Cases")
plot(p0, p1, layout=(1,2), legend=:topleft, xlabel="Days")

After 100 confirmed cases, the initial growth in cumulative confirmed counts are similar from day0 to ~day18. However, the suppression method employed by Korea has performed well in comparisons to other countries. Italy suffering the most. Iran seems to be slowing down in its growth, but not sure why...

After 50 deaths, Korea has the least growth in cumulative death counts. This growth seems to be largest for Spain > Italy > US > Iran. US is in the early stage and hope we don't see similar pattern as Italy. My friend Andrew Wang posted this recently on his facebook which may help alleviate this.

About 5% of Covid 19 patients end up requiring ICU care. These patient require early intubation and then progress to Acute Respiratory Distress Syndrome (ARDS). For my clinician friends I thought I'd share a few principles when treating ARDS. Remember it's all about lung protective strategies.

  1. Low tidal volume. 4-6mL/kg using predicted body weight.
  2. High PEEP! Use ARDSnet criteria and titrate PEEP together with FiO2.
  3. Permissive hypercapnea. Okay for PCO2 in the 60-70s.
  4. Low pH ~7.3. I'll even tolerate as low as 7.2
  5. SpO2 goals of 88-95%. Don't over oxygenate.
  6. Prone early! Coordinate with hospital/nursing staff because this tactic can actually save lives.

Korea is an outlier due to "Suppression" method of controlling COVID19 while others have been taking more of a "Mitigation" strategy until recently. More on this is written in this article.

Let's assume that 1) method of controlling virus was same across US, Italy, Spain, and Iran, and 2) US did well in identifying their "first case". The curves growing earlier than US suggests that other countries may have identified their "first case" late. This will result in delayed response towards the COVID19.

Approximately,

  • Spain: ~12-13 days
  • Italy: ~25 days
  • Iran: ~40 days

Taking day0 = date first death observed, the death growth is Spain > Italy > Iran == US > Korea. I think statistics are a bit early but the growth in US death counts may follow the pattern reported in Iran if not controlled effectively.

HOWEVER, the interpretation of above data is difficult because the start point of growth is different for each country based on day0 as first case seen. I think the results with day0 defined with thresholds are more comfortable to interpret (100 confirmed or 50 death cases).
